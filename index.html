Minesweeper

This project encompasses all the usual rules of the traditional minesweeper game. Users can customize their own board with the height, width and number of mines, entered as command line arguments. Users uncover a particular spot based on the coordinates specified.

The key functions-

  • Generated the board with user specified height,width and number of bombs to play the game.
  • Randomized the location of bombs using mt19937 (Mersenne Twister) algorithm.
  • Built the logic around uncovering the bombs.
  • Allow flagging of suspected bombs and ensure that these don't get clicked/uncovered.

From a secure coding perspective-

  • Care was taken to allocate and deallocate the memory for representing the board.
  • Used a better randomization function as compared to the usual rand().
  • Ensured the only valid inputs were accepted form the user.
  • All unexpected conditions were caught and handled appropriately.
